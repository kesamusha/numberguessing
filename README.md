Number Guessing Game Overview: This is a simple Python console game where the computer randomly selects a secret number between 1 and 200 and the player tries to guess it. The program provides hints if the guess is too high or too low until the correct number is guessed. Features: Random number generation using Python’s random module. Interactive user input with feedback after each guess. Loop continues until the player guesses correctly. Friendly messages to guide the player. How It Works: The program generates a random number between 1 and 10. The player is prompted to enter a guess. The program compares the guess with the secret number: If the guess is too low → prints “Too low! Try again.” If the guess is too high → prints “Too high! Try again.” If the guess is correct → prints “🎉 Congratulations! You guessed the correct number!” and ends the game. Requirements: Python 3.x No external libraries required (uses built-in random module). How to Run: Save the script as guessing_game.py. Open a terminal or command prompt. Run the program with: bash python guessing_game.py Follow the on-screen instructions to play.

Example Gameplay: Code 🎯 Welcome to the Number Guessing Game! I have chosen a number between 1 and 10. Enter your guess: 5 Too low! Try again. Enter your guess: 8 Too high! Try again. Enter your guess: 7 🎉 Congratulations! You guessed the correct number!
